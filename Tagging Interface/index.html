Remember:

  • When rating each dimension except metaphorical, each Tweet should be coded on the overall content of the Tweet, not just the portion of the tweet that is referencing AD or dementia. Do not consider original tweets if it is a reply.
  • Retweets should be coded based on the overall content of the tweet, including that original content that was retweeted.
  • "No" vs. "Can't tell": when it seems like a toss up between the two code as "No" if it is at all reasonable, otherwise, "Can't tell"
  • Not the Tweet we are looking for:if it is senile the song, or if they keyword is only in the username (e.g., @wild_demeted)
  • Metaphor: code very short ones such as "memory loss" as can't tell (when there is no other context). Comparison/simile using an idea that's not real counts as metaphor (e.g., she's a senile old bat). If its unclear, stick to use of the AD related keyword and if its being applied to someone/thing that doesn't literally have dementia it should be more likely to count as ridicule not necessarily metaphor>. Example of metaphor (e.g., big demented eyes, another demented selfie), example of NOT metaphor (e.g. senile geriatric president)
  • Personal experience: tweets need to explicitly reference a personal experience of the person posting the Tweet, not just an opinion or reference to someone else’s experience. Includes intentions, past tense, future tense (including hypothetical future, eg, if...then I would...)
  • Joke: If tweet mentions funny/humor, it counts as joke. Presence of emoticons (which we can't see) count as some context and provide a better reason to select "can't tell" versus "no". In other words, "no" is less a reasonable code in a very short tweet with emojis, for example: "i have short term memory loss " should be "can't tell" because "no" is less certain.
  • Ridicule: use the AD related keyword as a guide (e.g., the word demented is negative and probably indicates ridicule). --Exaggerations or hyperbole that are negative also count as ridicule. --Not ridicule when Alzheimer's Disease itself is the target of ridicule (e.g., "Alzheimer's Sucks).
  • Informative: Job postings in dementia related facilities count as informative. Pay attention to the type of source, if its a news article or scientific finding, it is informative. Ads for charity/fundraising events don’t count as informative unless it is a news story (headline) about this kind of event. In general, if it doesn’t have a link, its less likely to be informative (more likely to be personal opinion).
